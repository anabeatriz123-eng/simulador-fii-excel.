# Simulador de Investimentos em Fundos Imobiliários (FII) – Excel

## Sobre o projeto
Este projeto foi desenvolvido como parte de um desafio prático da DIO, com o objetivo de aplicar os conceitos de Excel aprendidos ao longo do curso.

A proposta consiste na criação de uma planilha de simulação de investimentos em Fundos Imobiliários (FIIs), auxiliando o usuário a entender melhor o comportamento do investimento ao longo do tempo.

## Objetivo
Criar uma ferramenta simples em Excel que permita:
- Simular investimentos mensais em FIIs
- Calcular o valor total investido
- Estimar o patrimônio acumulado
- Calcular os dividendos mensais

## Funcionalidades
- Entrada de valor investido mensal
- Definição do período de investimento
- Aplicação de taxa de rendimento mensal
- Cálculos automáticos

## Observação
Projeto desenvolvido para fins educacionais.
